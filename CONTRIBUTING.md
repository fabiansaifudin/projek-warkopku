# Panduan Kontribusi

Terima kasih ingin berkontribusi! Mohon ikuti panduan singkat ini.

1. Fork repo dan buat branch baru: `git checkout -b feat/nama-fitur`
2. Ikuti standar coding yang ada (formatting, linting).
3. Sertakan test untuk perubahan fungsional.
4. Push branch ke forkmu dan buat Pull Request ke `main`.
5. Pilih template PR dan isi semua bagian.
6. Minta review, perbaiki komentar jika diperlukan.

Standar commit:
- Gunakan pesan singkat: `type: deskripsi singkat`
- Contoh: `feat: tambah endpoint /users`

Jika ragu, buka issue terlebih dahulu untuk diskusi.
